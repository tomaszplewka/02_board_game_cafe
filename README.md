# Description 

**Spent quality time playing board games!**

- [x] Front-End part of the one and only Board Game Cafe project
- [x] Fully responsive website
- [x] Comes in four different flavours:
  - [x] Floats
  - [x] Flexbox
  - [x] Grid
  - [x] Bootstrap & PaperCSS
